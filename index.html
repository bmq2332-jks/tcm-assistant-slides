🤕 症狀深度剖析：頭痛

中醫辨證分型：肝陽上亢 vs 氣血虛弱

🔥 肝陽上亢型
  • 頭脹痛、眩暈耳鳴
  • 面紅目赤、心煩易怒
  • 舌紅苔黃、脈弦數
  • 因壓力大、熬夜、易怒
VS
💧 氣血虛弱型
  • 頭痛綿綿、勞累加重
  • 面色蒼白、心悸氣短
  • 舌淡苔薄白、脈細弱
  • 因體質虛弱、月經過多
📍 穴位
太衝（疏肝）· 風池（祛風）· 太陽（止痛）· 百會（昇陽）
💡 生活提醒
規律作息 · 冥想呼吸 · 忌熬夜 · 少吃辛辣

💪 肩頸腰背症候群

現代人久坐辦公室的常見困擾

🦒 肩頸酸痛
低頭族、電腦族第一殺手。風池 + 肩井 + 大椎，搭配米字操
🧊 五十肩
50 歲前後好發，肩關節沾黏。鐘擺運動 + 爬牆 + 毛巾牽拉
🤸 腰酸背痛
腰為腎之府。腎俞 + 委中 + 貓牛式 + 杜仲豬尾湯
⚡ 坐骨神經痛
腰腿放射痛。環跳 + 梨狀肌拉伸 + 俯臥伸展

🦵 四肢關節保健

膝為筋之府，肝主筋、腎主骨

🦵 膝蓋痛
退化、過重、運動傷害。犢鼻 + 靠牆半蹲 + 牛膝骨膠湯
🎾 網球肘
前臂勞損。曲池 + 手三里 + 前臂伸展 + 握力球訓練
🤚 手腕痛
腱鞘炎、媽媽手。陽谿 + 列缺 + 手腕伸展 + 人體工學配備
👣 足底筋膜炎
晨起第一步最痛。湧泉 + 太谿 + 足底滾球 + 毛巾勾腳

🌿 內科調理

脾胃為後天之本 · 心主神明

🍚 腸胃不適
  • 脾胃虛弱：食後腹脹、大便稀溏
  • 食滯胃脘：脹滿疼痛、噯腐吞酸
  • 穴位：足三里、中脘、天樞、內關
  • 食療：四神湯、陳皮生薑茶
  • 定時定量七分飽，細嚼慢嚥
VS
😴 失眠
  • 心脾兩虛：多夢易醒、心悸健忘
  • 心腎不交：心煩難眠、腰膝酸軟
  • 穴位：神門、安眠、三陰交、太衝
  • 食療：酸棗仁安神茶、桂圓蓮子湯
  • 睡前 1hr 不用 3C，固定作息
